Pelajari Lebih Lanjut

Tentang Program Keahlian RPL

Program Keahlian Rekayasa Perangkat Lunak (RPL) fokus pada pengembangan perangkat lunak, mulai dari analisis kebutuhan, perancangan sistem, coding, hingga pengujian aplikasi. Siswa mempelajari berbagai bahasa pemrograman modern, pembuatan website, aplikasi mobile (Android/iOS), serta pengelolaan database yang relevan dengan kebutuhan industri teknologi saat ini. Dengan dukungan laboratorium komputer berstandar industri, siswa dapat langsung mempraktikkan materi yang dipelajari.

Selain keterampilan teknis, program ini juga melatih kemampuan problem solving, logika pemrograman, serta kerja sama tim dalam proyek pengembangan software. Siswa dibimbing untuk berpikir kritis dan kreatif dalam menciptakan solusi digital yang bermanfaat bagi masyarakat maupun dunia usaha. Pembelajaran berbasis proyek (Project Based Learning) diterapkan agar siswa terbiasa dengan alur kerja pengembangan software yang sesungguhnya. Kurikulum selalu diperbarui mengikuti tren teknologi terbaru.

Lulusan RPL memiliki kompetensi sebagai programmer, web developer, mobile app developer, maupun IT support. Siswa dibekali kemampuan untuk beradaptasi dengan cepat terhadap perkembangan teknologi, sehingga siap bersaing di pasar kerja global maupun merintis startup teknologi sendiri. Program ini menjadi pilihan tepat bagi siswa yang memiliki minat tinggi di bidang teknologi informasi dan ingin menjadi bagian dari revolusi digital.

Keunggulan Program

Program RPL kami unggul dalam kurikulum berbasis industri, fasilitas lengkap, dan kerjasama dengan perusahaan teknologi terkemuka.

Kurikulum Berbasis Industri

Materi pembelajaran disesuaikan dengan kebutuhan industri software house dan startup teknologi, mencakup web development, mobile app, dan database management.

Lab Komputer Spesifikasi Tinggi

Fasilitas laboratorium dengan komputer berspesifikasi tinggi yang mendukung proses coding, rendering, dan simulasi server secara optimal.

Kemitraan Industri

Kerjasama dengan berbagai perusahaan IT untuk program magang, guru tamu, dan penyaluran kerja lulusan, memastikan relevansi kompetensi siswa.

Materi Pembelajaran

Memberikan dasar yang kuat dan pengalaman praktik nyata agar siswa siap menghadapi dunia kerja dan praktik profesional.

Semester 1-2: Dasar Pemrograman

  • Logika dan algoritma pemrograman
  • Dasar HTML, CSS, dan JavaScript
  • Pengenalan sistem komputer & jaringan dasar
  • Dasar desain grafis untuk antarmuka (UI)
  • Penggunaan tools pengembangan software

Semester 3-4: Pengembangan Aplikasi

  • Pemrograman Web Lanjut (PHP, Framework)
  • Dasar Pemrograman Berorientasi Objek (Serta Java/C#)
  • Basis Data (MySQL/SQL Server)
  • Pembuatan aplikasi desktop sederhana
  • Persiapan praktik industri (PKL)

Semester 5-6: Advanced & Mobile Dev

  • Pemrograman Aplikasi Perangkat Bergerak (Mobile/Android)
  • Pengembangan Sistem Informasi
  • Keamanan data dan testing aplikasi
  • Produk Kreatif dan Kewirausahaan (Startup)
  • Persiapan Uji Kompetensi Keahlian (UKK)

Prospek Karir

Lulusan RPL memiliki peluang karir yang sangat luas dan menjanjikan di era digital saat ini.

Web Developer

Membangun dan memelihara website serta aplikasi berbasis web untuk perusahaan atau klien.

Mobile App Developer

Merancang dan mengembangkan aplikasi untuk perangkat mobile on (Android/iOS).

Software Engineer

Mengembangkan, menguji, dan memelihara perangkat lunak sistem yang kompleks.

Database Administrator

Mengelola, menyimpan, dan mengamankan data perusahaan menggunakan sistem basis data.

IT Support/Consultant

Memberikan dukungan teknis dan solusi IT untuk operasional bisnis atau organisasi.
